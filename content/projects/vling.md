## 개요

Vling은 유튜브 크리에이터와 광고주를 연결하는 마케팅 플랫폼입니다. 글로벌 서비스 런칭을 위한 UI/UX 개편, 구글 애드센스 기반 광고 수익화 모델 구축, 결제 시스템 TypeScript 마이그레이션 및 장애 모니터링 체계 구축, 유튜버 광고 역제안 서비스 기획·개발 등 서비스 전반의 프론트엔드 개발을 담당했습니다.

## 담당 업무

- 글로벌 런칭을 위한 메인 페이지 리뉴얼 및 수익 계산기 글로벌화
- 구글 애드센스 기반 광고 수익화 모델 구축 및 A/B 테스트 운영
- 결제 코드 TypeScript 마이그레이션 및 함수형 리팩토링
- Sentry + Slack 연동 실시간 장애 모니터링 체계 구축 및 사내 확산
- 모바일 결제 UX 프로세스 개선
- 유튜버 광고 역제안 서비스 기획 및 MVP 개발

## 주요 기능

- 콘텐츠 중심 메인 페이지 리뉴얼 및 글로벌 RPM/CPM 기반 수익 계산기
- 구글 애드센스 광고 컴포넌트 추상화 및 구독 유저 광고 비노출 정책
- 실시간 결제 장애 모니터링 및 유사 장애 사례 추천 슬랙 봇
- 모바일 결제 자동 스크롤 및 필수 입력 강조 UX
- 유튜버 채널 인사이트 기반 광고 역제안 서비스

## 기술적 도전과 해결

### 글로벌 서비스 런칭을 위한 UI/UX 개편

**문제:** 해외 유저 확보를 위해 메인 페이지의 글로벌 UX 개편이 필요했습니다. 기존 검색 중심이 아닌 콘텐츠 탐색 중심의 구조 전환과 해외 시장에 맞는 수익 계산기 기능이 부재했습니다.

**해결:** 메인 페이지를 콘텐츠 중심 구조로 리뉴얼하고 카드 UI·인터랙션을 적용하여 사용성을 개선했습니다. 랜딩 방식을 검색 중심으로 전환하여 상단 영역 이용률을 높이고, 글로벌 RPM/CPM 기준에 맞춘 수익 계산기 기능을 신규 개발하여 해외 사용자 접근성을 강화했습니다.

### 구글 애드센스 기반 광고 수익화 모델 구축

**문제:** 비회원 유저 대상 수익화 구조가 부재했습니다. 애드센스 광고 삽입 시 스타일 깨짐, 스크롤 충돌 등의 이슈가 발생하고, 구독 유저의 UX를 저해하지 않으면서 수익화를 병행해야 했습니다.

**해결:** 애드센스 스타일 깨짐·스크롤 충돌 등 광고 삽입 이슈를 자동 감지·복원하는 로직을 개발하여 운영 리스크를 최소화했습니다. 광고 컴포넌트를 추상화하여 신규 페이지 적용 시간을 40% 단축하고 일관된 재사용 구조를 확립했습니다. 구독 유저 대상 광고 비노출 정책을 적용하고, 전면 광고 대상 A/B 테스트를 운영하여 CTR·RPM·가입 이탈률 분석 기반으로 수익 효율을 극대화했습니다.

### 결제 시스템 안정화 및 장애 모니터링 체계 구축

**문제:** 결제 장애 발생 시 실시간 모니터링 체계가 부재해 고객 문의와 서버 로그를 통해서만 장애를 인지할 수 있었습니다. 레거시 JSX 기반 결제 코드로 타입 안정성이 낮았고, 타사 프로모션 진행 시 인증·결제 예외 처리를 개별 분기로 추가해 사이드 이펙트가 증가했습니다. 또한 모바일 결제 과정에서 24% 이상의 중도 이탈이 발생하고 있었습니다.

**해결:** 결제 코드 전반을 TypeScript로 마이그레이션해 정적 타입 기반 안정성을 확보하고, 공통 결제 로직을 모듈화하여 함수형 기반으로 리팩토링해 결제 수단 확장 시 영향 범위를 최소화했습니다. Sentry와 Slack 알림을 연동한 실시간 장애 모니터링 체계를 구축하고, 과거 유사 장애 포스트모템 문서를 추천하는 슬랙 봇을 구현하여 대응 속도를 개선했습니다. 모바일 결제 UX에는 자동 스크롤 처리 및 필수 입력 강조 UI를 적용하여 단계별 입력 구조를 재정렬했습니다.

### 유튜버 광고 역제안 서비스 기획 및 개발

**문제:** 유튜버 대상 서비스가 부재하여 크리에이터 유입과 플랫폼 활성화에 한계가 있었습니다.

**해결:** 유튜버 인터뷰 및 요구사항 분석을 통해 서비스 기획 방향을 정의하고, 채널 데이터 기반 광고 역제안 프로세스를 설계하여 초기 MVP를 개발했습니다. 온보딩 프로젝트로 기획·개발한 서비스가 실제 고도화되어 '내 채널 리포트' 신규 서비스로 출시되었습니다.

## 성과

- 메인 페이지 개편 후 가입 전환율 22.5% 상승(9.3% → 11.4%), 상단 영역 전환율 2.5배 향상(2.2% → 5.36%)
- 광고 수익화 구조 도입으로 런칭 한 달 만에 US$400 달성, 광고 컴포넌트 적용 시간 40% 단축(2.5시간 → 1.5시간)
- 에러 대응 시간 50% 단축(8시간 → 4시간), 결제 오류 고객 문의 20% 감소
- 모바일 결제 이탈률 22% 감소(24% → 18.72%)
- 유튜버 역제안 MVP가 '내 채널 리포트' 서비스로 고도화·출시, 유튜버 전용 요금제 런칭에 기여
