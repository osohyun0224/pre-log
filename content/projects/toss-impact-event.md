## 개요

토스임팩트 이벤트 서비스는 사용자가 닉네임을 입력하고 아트워크를 선택하면 개인화된 카드 이미지를 생성하여 다운로드할 수 있는 캠페인 페이지입니다. 메타 광고 기반의 대고객 캠페인으로 운영되었습니다.

## 담당 업무

- 이벤트 서비스 전체 프론트엔드 개발 (닉네임 입력 → 아트워크 선택 → 카드 생성/다운로드)
- 인트로 페이지 마우스 hover 기반 아트워크 이미지 인터랙션 구현
- HTML2Canvas 기반 이미지 합성 및 다운로드 기능 구현
- SNS 인앱 브라우저 환경 대응 및 성능 최적화

## 주요 기능

- 인트로 페이지에서 마우스 hover 시 아트워크 이미지가 부드럽게 전환되는 인터랙션
- 닉네임 입력 후 아트워크를 선택하여 개인화된 카드 이미지 생성
- 생성된 카드 이미지 다운로드
- 카드 생성 후 hover 기반 인터랙션 효과 (PC)

## 기술적 도전과 해결

### 아트워크 카드 이미지 합성

**문제:** 사용자가 선택한 아트워크에 닉네임 텍스트를 합성하여 하나의 카드 이미지로 생성해야 했습니다. 카드 생성 이후 화면에서의 hover 인터랙션도 버벅임 없이 부드럽게 동작해야 했습니다.

**해결:** HTML2Canvas 기반 이미지 합성 방식을 적용하여 선택한 아트워크에 닉네임을 합성한 결과 카드를 생성하도록 구현했습니다. 카드 생성 이후 hover 기반 인터랙션 효과가 버벅임 없이 부드럽게 진행되도록 성능을 최적화했습니다.

### 인트로 페이지 아트워크 인터랙션

**문제:** 인트로 페이지에서 마우스 hover 시 여러 장의 아트워크 이미지가 부드럽게 전환되어야 했습니다. 이미지 전환 시 로딩 지연이나 깜빡임이 발생하면 인터랙션의 완성도가 크게 떨어지는 문제가 있었습니다.

**해결:** 이미지 프리페치(prefetch) 방식을 적용하여 페이지 진입 시점에 모든 아트워크 이미지를 미리 로드하도록 구현했습니다. 이를 통해 hover 시 이미 캐싱된 이미지가 즉시 표시되어 전환 지연 없이 부드러운 인터랙션을 구현할 수 있었습니다.

### SNS 인앱 브라우저 이미지 해상도 저하

**문제:** 인스타그램 등 SNS 인앱 브라우저에서 텍스트가 조합된 카드 이미지의 해상도가 저하되어 품질이 떨어지는 문제가 발생했습니다.

**해결:** Canvas scaling 조정 방식을 적용하여 각 모바일 기기별 디바이스 픽셀 비율에 맞게 캔버스 크기를 조정했습니다. 이를 통해 전 모바일 기기의 인앱 브라우저에서 고화질 이미지 품질을 유지한 채 다운로드할 수 있도록 구현했습니다.

## 성과

- 대고객 트래픽이 발생한 메타 광고 기반 캠페인에서 이미지 생성/다운로드 기능이 안정적으로 동작
- 이미지 프리페치를 통한 인트로 페이지 hover 인터랙션의 지연 없는 부드러운 전환 구현
- 전 모바일 기기 인앱 브라우저에서 고화질 이미지 품질 유지
