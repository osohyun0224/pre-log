## 개요

토스 비터스윗 마켓은 소상공인과 자영업자의 지속가능한 성장을 함께 고민하는 토스임팩트의 활동입니다. 토스임팩트는 기술과 서비스의 혁신을 넘어, 세상에 긍정적인 변화를 만들기 위한 토스의 노력과 실천을 의미합니다.

## 담당 업무

- 비터스윗마켓 서비스 웹 프론트엔드 개발
- 브랜드별 고해상도 이미지 최적화 및 UI 인터랙션 구현

## 주요 기능

- 브랜드별 디저트 상세 페이지 탐색
- 화면 전환 인터랙션을 통한 단계적 콘텐츠 탐색 경험
- 고해상도 이미지 기반의 비주얼 중심 레이아웃

## 기술적 도전과 해결

### 이미지 최적화

**문제:** 브랜드별 스토리컷, 제품컷, 현장컷 등 고해상도 이미지가 다수 포함되어 페이지 로딩 성능에 영향을 줄 수 있었습니다.

**해결:** 고해상도 이미지를 WebP 포맷으로 재구성하여 기존 대비 4배 해상도를 유지하면서도 평균 45~60% 용량 절감을 달성했습니다.

### 페이지 전환 인터랙션

**문제:** 디저트 상세 페이지의 상하 이미지 전환 과정에서 깜빡임이나 로딩 지연이 발생하여 사용자 경험이 끊기는 문제가 있었습니다.

**해결:** 이미지 로딩 타이밍과 렌더링 순서를 조정하여 자연스럽게 전환되는 UI 인터랙션을 구현했습니다. 사용자가 브랜드 콘텐츠를 단계적으로 탐색할 수 있도록 경험을 설계했습니다.

## 성과

- 이미지 용량 평균 45~60% 절감 (4배 해상도 유지)
- 페이지 전환 시 깜빡임 및 로딩 지연 제거
