## 개요

토스임팩트는 토스의 사회적 가치 실현 활동을 소개하는 공식 웹사이트입니다. 메인 히어로 섹션에 대용량 비디오를 끊김 없이 재생하기 위한 최적화 작업을 수행했습니다.

## 담당 업무

- 토스임팩트 공식 웹사이트 프론트엔드 개발
- 메인 히어로 섹션 비디오 재생 최적화

## 주요 기능

- 메인 히어로 섹션 끊김 없는 비디오 재생
- 토스의 사회적 가치 실현 활동 소개 페이지

## 기술적 도전과 해결

### 메인 히어로 비디오 끊김 없는 재생 최적화

**문제:** 메인 히어로 섹션에 대용량 비디오가 포함되어 있어, 네트워크 환경에 따라 버퍼링이 발생하거나 재생이 끊기는 문제가 있었습니다. 페이지 진입 시 비디오가 즉시 자연스럽게 재생되어야 브랜드 메시지 전달력을 유지할 수 있었습니다.

**해결:** 비디오 프리로드 전략을 적용하여 페이지 진입 시점에 비디오 리소스를 미리 로드하고, 충분한 버퍼가 확보된 후 재생을 시작하도록 구현했습니다. 비디오 포맷 최적화와 적절한 압축을 통해 파일 크기를 줄이면서도 화질을 유지하여 다양한 네트워크 환경에서 끊김 없는 재생을 보장했습니다.

## 성과

- 메인 히어로 비디오 끊김 없는 재생으로 브랜드 메시지 전달력 확보
- 다양한 네트워크 환경에서 안정적인 비디오 재생 경험 제공
